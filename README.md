# 🧩 Incidencias Project

Sistema de gestión de incidencias desarrollado como proyecto personal para practicar arquitectura full stack moderna.

---

## 📌 Descripción

Este proyecto está dividido en varios repositorios independientes para simular una arquitectura real de producción:

- Frontend SPA con Angular
- Backend API REST con Spring Boot
- Backend MVC con Spring (próximamente)

---

## 🔗 Repositorios

### 🖥️ Frontend Angular
👉 https://github.com/JuanMelgar01/incidencias-angular

### ⚙️ Backend API REST (Spring Boot)
👉 https://github.com/JuanMelgar01/incidencias-spring-api

### 🧪 Backend Spring MVC (próximamente)

## 🎯 Objetivo del proyecto

- Practicar desarrollo full stack real
- Implementar autenticación JWT
- Centralizar manejo de errores
- Comparar arquitectura SPA vs MVC
